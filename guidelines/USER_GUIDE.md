# FlipClock 사용자 가이드

FlipClock에 오신 것을 환영합니다! 이 가이드는 앱을 최대한 활용하는 방법을 설명합니다.

---

## 📱 앱 설치

### 웹 브라우저에서
1. `flipclock.vercel.app`에 접속
2. 주소창 옆의 **설치 아이콘** (또는 메뉴의 "앱 설치") 클릭
3. 홈 화면에 추가됨

### 장점
- 네트워크 속도 개선
- 더 빠른 실행
- 오프라인에서도 사용 가능

---

## ⏰ 시계 화면 사용법

### 기본 조작

| 동작 | 설명 |
|------|------|
| **길게 누르기** (150ms 이상) | 시계를 드래그하여 위치 이동 가능 |
| **더블탭** | 시계를 원래 위치로 리셋 + 전체화면 토글 |
| **핀치 줌** (2손가락) | 시계 크기 조절 (0.3x ~ 2.5x) |
| **좌측 스와이프** | 타이머 화면으로 전환 |
| **상향 스와이프** | 설정 화면 열기 |
| **하향 스와이프** | 설정 화면 닫기 |

### 디스플레이 정보

**시계 표시:**
- 시간 (12시간 형식)
- 분
- 초 (클릭 토글 가능)

**좌측 타일 상단:** 연월일 (YYYY.MM.DD)  
**좌측 타일 하단:** AM/PM (클릭 토글 가능)

**우측 타일 상단:** 요일 (월요일, 화요일 등)  
**우측 타일 하단:** 초 표시/숨김 토글

### 터치 피드백
- 길게 누르면 진동 피드백 (지원 기기에만 해당)
- 드래그 중일 때 "드래그로 이동 · 더블탭으로 리셋" 안내 표시

## ⚙️ 설정 화면

메인 화면에서 **상향 스와이프**하면 설정 화면이 열립니다.

### 색상 설정

#### 프리셋 테마 (6가지)
- **기본**: 청록색 + 분홍 배경
- **다크**: 검은색 + 진회색 배경
- **블루**: 진파랑색 + 파랑 배경
- **그린**: 진초록색 + 초록 배경
- **퍼플**: 자주색 + 보라색 배경
- **오렌지**: 주황색 + 주황 배경

테마를 클릭하면 즉시 적용됩니다.

#### 커스텀 색상
- **타일 배경색**: 시계 타일의 배경색
- **텍스트 색상**: 시간/분/초의 글자색
- **앱 배경색**: 화면 전체의 배경색

각 항목 옆의 **색상 팔레트 아이콘**을 클릭하여 원하는 색상 선택

### 배경 이미지 설정

**배경 이미지 테마:**
- 9개의 프리셋 이미지 제공
- 테마 클릭 시 즉시 적용
- 선택된 이미지는 앱 재시작 후에도 유지됨

**이미지 변경 방법:**
1. "배경 이미지" 섹션에서 원하는 이미지 선택
2. "테마 저장" 클릭하여 적용

## ✨ 파티클 효과

### 파티클 효과란?
메인 화면에 떠내려가는 입자 애니메이션 효과입니다.

### 효과 종류

| 이름 | 모양 | 색상 | 설명 |
|------|------|------|------|
| 🌸 벚꽃 | 하트/타원 | 분홍색 | 봄날 벚꽃이 흩어지는 느낌 |
| 🍃 나뭇잎 | 타원 | 초록색 | 신선한 초록 나뭇잎 |
| 🍂 낙엽 | 타원 | 주황/빨강 | 가을 낙엽이 떨어지는 느낌 |
| ❄️ 눈꽃 | 눈 결정 | 하얀색 | 겨울 눈이 소복이 내리는 느낌 |
| 🌧️ 비 | 타원 | 파란색 | 빗방울이 떨어지는 느낌 |

### 파티클 효과 활성화

1. 설정 화면 하단의 **"파티클 효과"** 찾기
2. **ON** 토글 활성화
3. 원하는 효과 선택 (벚꽃, 나뭇잎, 낙엽, 눈꽃, 비 중 1개)
4. 메인 화면으로 돌아가면 애니메이션 표시

### 주의사항
- 한 번에 1가지 효과만 활성화 가능
- 효과 OFF 상태에서는 애니메이션 미표시
- 배터리 사용량 증가 (필요시 OFF 권장)

## 🎨 시계 디스플레이 조절

### 투명도 조절
- **위치**: 설정 화면 → "시계 투명도" 슬라이더
- **범위**: 0% (완전 투명) ~ 100% (완전 불투명)
- **사용 예**: 배경 이미지를 더 잘 보고 싶을 때

### 시계 위치 저장
- 메인 화면에서 시계를 **길게 누르고 드래그**하여 원하는 위치로 이동
- 앱을 종료했다가 다시 열어도 **위치가 유지됨**
- **더블탭**으로 원래 위치로 리셋 가능

## 🔋 화면 관련 설정

### 화면 항상 켜짐 (WakeLock)
- **설명**: 기기의 자동 잠금 기능을 비활성화
- **사용 예**: 침대 옆에 시계처럼 사용할 때
- **주의**: 배터리 사용량 증가

### 화면 보호 모드 (Screen Saver)
- **설명**: 화면 번인(부모강) 방지를 위해 픽셀을 주기적으로 이동
- **사용 예**: OLED 디스플레이에서 장시간 같은 이미지 표시 시

### 권장 조합
- **충전 중**: 화면 항상 켜짐 ON + 파티클 효과 ON
- **장시간 자리 비움**: 화면 보호 ON + 화면 항상 켜짐 OFF
- **OLED 기기**: 잠자 세로 고 ON 권장 (픽셀 시프트)

- **모든 설정은 자동으로 저장됨**
- 색상, 테마, 파티클 효과, 시계 위치 등

일부 설정은 **"테마 저장" 버튼**으로 명시적 저장 필요:

- 배경 이미지

## ❓ 자주 묻는 질문 (FAQ)

### Q: 파티클 효과가 보이지 않습니다.
**A:** 
1. 설정에서 파티클 효과가 **ON**으로 설정되었는지 확인
2. 입자 타입 (벚꽃, 나뭇잎 등)이 선택되었는지 확인
3. 메인 화면으로 돌아갔는지 확인
4. 캐시 삭제 후 새로고침 시도

### Q: 시계 위치가 저장되지 않습니다.
**A:**
- 브라우저가 localStorage를 차단하지 않는지 확인
- 비공개/시크릿 모드가 아닌지 확인
- 앱을 다시 설치 시도

### Q: 화면이 깜빡입니다.
**A:**
- 모바일 기기에서 발생할 수 있음
- "화면 보호" OFF 시도
- 파티클 효과 OFF 시도
- 다른 앱 종료 후 메모리 확보

### Q: 배터리가 빨리 소모됩니다.
**A:**
- "화면 항상 켜짐" OFF
- "파티클 효과" OFF
- "화면 보호" OFF
이 중 불필요한 기능을 비활성화하세요.
