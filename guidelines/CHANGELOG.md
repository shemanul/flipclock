# FlipClock Changelog

모든 주목할 만한 변경사항이 이 파일에 문서화됩니다.

## [v0.6.0] - 2026-04-22

### ✨ 새로운 기능

#### 파티클 효과 시스템 (Particle Effect System)
- 메인 화면에 다양한 애니메이션 입자 효과 추가
- **5가지 입자 타입 지원:**
  - 🌸 **벚꽃**: 분홍색 하트/타원 모양
  - 🍃 **나뭇잎**: 초록색 타원 모양
  - 🍂 **낙엽**: 주황색/빨간색 타원 모양
  - ❄️ **눈꽃**: 하얀색 눈 결정 모양 (6개 가지)
  - 🌧️ **비**: 파란색 타원 모양

- 설정 화면에서 파티클 효과 ON/OFF 토글
- 원하는 입자 타입 선택 (한 번에 1가지만 활성화 가능)

#### 시계 디스플레이 조절
- **투명도 조절**: 시계의 투명도를 0~100% 범위에서 조절 가능
- **위치 저장**: 사용자가 조정한 시계 위치가 앱 재시작 후에도 유지됨

#### 배경 이미지 테마
- 사용자가 선택한 배경 이미지가 설정과 함께 저장됨
- 6개의 프리셋 배경 이미지 테마 제공
- 테마 저장 시 이미지도 함께 저장

### 🐛 버그 수정
- 파티클 캔버스 z-index 문제 수정 (canvas를 타일 위에 렌더링)
- ClockScreen에서 정의되지 않은 변수 참조 제거
- 이전 벚꽃 배경 UI 제거 (새로운 파티클 시스템으로 통합)

### 🔧 기술적 개선
- GPU 가속을 통한 성능 최적화 (CSS: contain, transform3d, willChange)
- 모바일 터치 이벤트 최적화 (requestAnimationFrame 활용)
- Canvas 기반 고성능 입자 애니메이션 렌더링
