# FlipClock — 사용 방법

> 플립 시계 PWA 앱 사용 가이드

---

## 설치 방법

1. 크롬 브라우저로 접속: `https://flipclock-eight.vercel.app`
2. 주소창 오른쪽 **⋮ 메뉴** → **"홈 화면에 추가"**
3. 설치 완료 → 홈화면 아이콘으로 실행

> 💡 반드시 **홈화면 아이콘(PWA)**으로 실행해야 모든 기능이 정상 동작해요.

---

## 화면 구성

```
┌──────────────────────────────────────┐
│                                      │
│    ┌──────────┐    ┌──────────┐     │
│    │ 26.04.21 │    │  화요일  │     │
│    │          │    │          │     │
│    │    10    │    │    30    │     │
│    │          │    │          │     │
│    │ AM       │    │       45 │     │
│    └──────────┘    └──────────┘     │
│                                      │
└──────────────────────────────────────┘

시 타일: 날짜 / 시각 / AM·PM
분 타일: 요일 / 분 / 초
```

## 제스처 조작

### 화면 전환

| 제스처 | 동작 |
|--------|------|
| 위로 스와이프 | 설정 화면 열기 |
| 아래로 스와이프 | 설정 화면 닫기 |
| 왼쪽 스와이프 | 타이머 화면으로 이동 |
| 오른쪽 스와이프 (타이머에서) | 시계 화면으로 복귀 |

### 시계 조작

| 제스처 | 동작 |
|--------|------|
| **AM/PM 탭** | AM ↔ PM 표시 토글 |
| **초 탭** | 초 표시 토글 |
| **두 손가락 핀치** | 시계 크기 조절 |
| **길게 누르기(0.4초) + 드래그** | 시계 위치 이동 (진동으로 시작 알림) |
| **더블탭** | 크기·위치 초기화 + 상태바 토글 |

> 💡 **더블탭** → 상태바가 나타나면 홈버튼이나 앱 스위처로 앱 종료 가능

---

## 타이머 사용법

1. 왼쪽으로 스와이프 → 타이머 화면
2. **▶** 버튼 → 시작
3. **⏸** 버튼 → 일시정지
4. **↺** 버튼 → 리셋
5. 오른쪽 스와이프 → 시계 화면 복귀

---

## 설정 화면

위로 스와이프하면 설정 화면이 열려요.

### 테마 & 색상

**프리셋 테마 선택**
- 기본 / 다크 / 블루 / 그린 / 퍼플 / 오렌지 중 탭하여 선택

**색상 직접 지정**
1. 색깔 칩(■) 탭 → 색상 슬라이더 펼침
2. 슬라이더 조작:
   - **색조** — 빨강/노랑/초록/파랑 등 기본 색상
   - **채도** — 선명함 (왼쪽: 회색, 오른쪽: 선명)
   - **밝기** — 밝기 (왼쪽: 검정, 오른쪽: 흰색)
3. HEX 코드 직접 입력 가능 (예: #FF5733)
4. 좌측 미리보기에서 즉시 확인

**테마 저장/삭제**
- "현재 색상을 테마로 저장" 버튼 → 최대 6개 저장
- 저장된 테마 우상단 **×** 탭 → 삭제

**배경 이미지**
- "이미지 업로드" → 갤러리에서 선택
- 🗑️ 버튼 → 이미지 제거

---

### 글씨체 & 크기

| 설정 | 조작 방법 |
|------|----------|
| 폰트 선택 | ▲▼ 화살표로 변경 (20종) |
| 글자 크기 | − / 숫자 직접입력 / + (길게 누르면 연속 변경) |

- **메인 글자**: 시계 숫자 (시, 분)
- **부가 정보**: 날짜, 요일, AM/PM, 초

---

### 화면 설정

| 설정 | 설명 |
|------|------|
| **화면 보호** | 3초마다 화면이 ±4px 이동 → OLED 번인 방지 |
| **입자 픽셀 새로고침** | 파티클 50개 + 눈꽃 30개 + 8초마다 픽셀 이동 |
| **화면 켜짐 유지** | 충전 중 화면이 꺼지지 않게 유지 (WakeLock) |
| **벚꽃 배경** | 봄날 벚꽃잎 60개가 휘날리는 애니메이션 |

**💡 권장 조합**

| 상황 | 추천 설정 |
|------|----------|
| 충전 중 탁상 시계로 사용 | 화면 켜짐 유지 ON |
| 오랫동안 자리 비울 때 | 화면 보호 ON |
| OLED 폰 (갤럭시 등) | 입자 픽셀 새로고침 ON |
| 봄 분위기 연출 | 벚꽃 배경 ON |

> ⚠️ 화면 보호 ON이면 화면 켜짐 유지는 자동으로 비활성화돼요.

## 자주 묻는 질문

**Q. 시계가 너무 작아요.**
A. 두 손가락으로 화면을 벌려서 크게 키울 수 있어요.

**Q. 시계를 화면 가운데가 아닌 다른 위치에 놓고 싶어요.**
A. 시계를 0.4초 길게 누른 후(진동 느낌) 드래그하면 이동해요. 더블탭하면 원래 위치로 돌아와요.

**Q. 앱을 종료하고 싶어요.**
A. 더블탭 → 상태바가 나타나면 홈버튼이나 앱 스위처에서 종료하세요.

**Q. 번인이 걱정돼요.**
A. 화면 보호 + 입자 픽셀 새로고침을 모두 ON하면 픽셀 시프트 효과로 번인을 최소화해요.

**Q. 벚꽃이 안 나타나요.**
A. 설정 → 화면 설정 → 벚꽃 배경을 ON으로 켜주세요.

앱이 자동으로 최신 버전을 감지해요. 수동으로 업데이트하려면:

2. **Clear site data** 클릭
3. 앱 재실행
